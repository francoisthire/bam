Module Bam

This library can be used to write property-based tests.

The library is designed so that most of the time, all the functions needed are contained within the Std module, in particular, it defines basic generators that can be used to write the generators you want.

The library allows you to define easily new generators using a monad-like structure. Such a monad behaves well with respect to shrinking. This makes shrinking predictable and facilitates the debugging.

Also, the library exports different shrinking strategies to ease the search for a minimal counter-example.

Modules Gen can be used to define ad-hoc shrinkers.

If you want to understand how the shrinking work in this library, feel free to read our our primer on shrinking.
